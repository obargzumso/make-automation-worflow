# Make.com Automation Workflow

A no-code automation workflow built with Make.com for handling form submissions.

## What it does

- Receives form submissions via webhook
- Sends email notification to the client
- Logs submission data to Google Sheets
- Fully automated — zero manual work required

## Tools Used

- Make.com (formerly Integromat)
- Google Sheets
- Gmail / SMTP
- Webhook trigger

## Use Case

Built for a client who needed to automatically capture leads from a contact form, notify the team by email, and store all submissions in a spreadsheet for follow-up.

## Workflow Steps

1. Form submitted → webhook triggered
2. Make.com parses name, email, message
3. Email notification sent automatically
4. Data logged to Google Sheets with timestamp
